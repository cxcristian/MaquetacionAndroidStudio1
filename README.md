# FirstProject3114227

Primer proyecto Android con Jetpack Compose - Maquetación Android Studio.

## Requisitos

- Android Studio Hedgehog (2023.1.1) o superior
- JDK 17 o superior
- Gradle 8.x
- Kotlin 2.0.x

## Cómo importar el proyecto

1. **Clonar el repositorio**
   ```bash
   git clone https://github.com/cxcristian/MaquetacionAndroidStudio1.git
   ```

2. **Abrir en Android Studio**
   - File → Open → Seleccionar la carpeta del proyecto clonado
   - Esperar a que Android Studio sincronice con Gradle (puede tomar unos minutos la primera vez)

3. **Ejecutar**
   - Seleccionar un dispositivo/emulador
   - Hacer clic en **Run** (▷) o usar `Shift + F10`

## Solución de problemas

- Si Gradle falla, ve a **File → Invalidate Caches → Invalidate and Restart**
- Si hay errores de SDK, abre **File → Project Structure → SDK Location** y verifica la ruta del SDK de Android
- Asegúrate de tener instalado **Android SDK Platform 36** (API 36) desde **SDK Manager**
